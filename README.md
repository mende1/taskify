# Taskify

Just another To-Do app for learning Hotwire in a Rails app.

Ruby-version: 3.2.2

Rails-version: 7.1.2

Following the instrucions of the site https://courses.writesoftwarewell.com/p/hotwire-handbook

## Run the Project

Install dependencies:

```
bundle install
```

And run the projet by:

```
./bin/dev
```
